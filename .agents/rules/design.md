# Hermes Chat - Design System

## 1. Visão Geral
**Estilo:** Neo-brutalismo Moderno.
**Conceito:** Alto contraste, bordas pretas espessas, sombras rígidas (hard shadows) e cores saturadas. O design foca na funcionalidade crua com uma estética "pop".

## 2. Paleta de Cores
- **Primária (AI):** `#5461EE` (Royal Blue)
- **Secundária (Destaques/Botões):** `#5EE79D` (Pastel Green)
- **Ação/Aviso:** `#FF8702` (Flush Orange)
- **Fundo Principal:** `#1C1C1C` (Dark Charcoal)
- **Texto/Cards Usuário:** `#FFFFFF` (Pure White)
- **Bordas/Sombras:** `#000000` (Pure Black)

## 3. Tipografia
- **Fonte Principal:** Space Grotesk
- **Estilo:** Bold ou Extra-Bold para títulos e botões.
- **Peso das Bordas:** 3px a 4px em todos os elementos de interface.

## 4. Componentes Chave
- **Cards de Mensagem:**
    - IA: Fundo Azul, texto branco, borda preta 3px, sombra preta 4px (bottom-right).
    - Usuário: Fundo Branco, texto preto, borda preta 3px, sombra preta 4px.
- **Botões:**
    - Devem ter um efeito de "clique" físico, onde a sombra desaparece e o botão se desloca 4px para baixo e para a direita.
- **Inputs:**
    - Fundo branco ou cinza muito escuro com contorno preto grosso.

## 5. Layout (Web Desktop)
- **Sidebar:** 280px de largura fixa.
- **Main Chat:** Fluído, com largura máxima de leitura (approx. 800px) centralizada para as mensagens.
